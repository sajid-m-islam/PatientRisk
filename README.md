# Patient Risk

A full-stack web application that assesses user health risks based on biographical, biometric, and dietary information. The application features secure user authentication, real-time risk calculation, and historical data tracking.

## Live Demo

You can try out the application here:
https://patientrisk.onrender.com/

## Tech Stack

-   **Frontend:** React (Vite), JavaScript
-   **Backend:** Python (FastAPI)
-   **Database & Auth:** Supabase
-   **Machine Learning:** Pandas, NumPy, TensorFlow
-   **Deployment:** Render

## Key Features

-   **Secure Authentication:** User sign-up and login managed via Supabase Auth (with email verification).
-   **Health Risk Algorithm:** Custom Python logic that calculates risk scores through a neural network based on health information.
-   **Interactive Dashboard:** React-based UI for users to input data and view immediate results.
-   **Data Persistence:** Saves user health profiles and risk assessment history to a PostgreSQL database.
